# Edpuzzle-Solver
A Python Pyautogui-based program that automatically solves Edpuzzles using image detection and AI.

# DISCLAIMER
This program utilizes a very rudamentary method of solving edpuzzles. This should only be used as a secondary way to complete them. For a better method, please visit https://edpuzzle.hs.vc for a more user-friendly application.
Additionally, this program is not garunteed to get the correct answer to an edpuzzle, especially if the questions are based on the specific video that you are watching and not a fundamental concept that the video is teaching.

# Requirements
This program requires Python 3.x and PIP to function.

# Installation
Simply download the code as a .zip folder, extract it, and run Edpuzzle.py with Python 3. The dependencies should install automatically.
